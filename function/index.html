                                                                                                          # array_reduce

                                                                                                          从左到右为每个数组元素执行一次回调函数，并把上次回调函数的返回值放在一个暂存器中传给下次回调函数，并返回最后一次回调函数的返回值。

                                                                                                          array_reduce(array, callback(accumulator, current_value[, index[, array]])[, initial_value])

                                                                                                          参数

                                                                                                          返回值
